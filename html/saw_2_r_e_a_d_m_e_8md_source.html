5 ## Dependencies
6 
7 You will need a version of SAW (https://saw.galois.com/).
8 
9 The official releases are quite old at this point.
10 The simplest method is to use a nightly (https://saw.galois.com/builds/nightly/). All proofs I have currently run are
11 performed using the saw-0.2-2018-09-05-MacOSX-6 nightly build.
12 It is also possible to build SAW from source.

14 ## Solvers
15 
16 SAW has several automated tactics. The ProofScript values abc and z3 select the ABC and Z3 theorem provers,
17 respectively, and are typically good choices. In addition to these, the boolector, cvc4, mathsat, and yices provers
18 are available. These do not need to be installed and are internal.
19 
20 SAW supports more generic interfaces to other arbitrary theorem provers supporting specific interfaces, but I don't
21 think this will be necessary.

23 ## Code Structure
24 
25 There are five .cry files and one .saw file.
26 
27  type_def.cry contains the type declarations for the Cryptol types as well as constants.
28 
29  helper_functions.cry contains helper functions to reduce code bloat due to common subroutines.
30 
31  transition_functions.cry contains the Cryptol specification of the state machine transition functions.
32 
33  state_machine.cry contains the Cryptol specification of the state machine, conditions for the process loop to
34  continue, and buffer update checks.
35 
36  proofs.cry contains a set of progress properties verifiably with a SAT-solver.
37 
38  state_machine.saw contains the harness from SAW to run the proofs specified in proofs.cry
39 
40 ## How to Run
41 
42 To run state_machine.saw:
43 
44  $PATH_TO_SAW_DIR/bin/saw state_machine.saw

46 ## Equivalence Proof
47 
48 SAW can also be leveraged to prove the equivalence between the Cryptol specification and the underlying C code. These
49 proofs are currently a work in progress but there will be a seperate pull request containing them. The equivalence
50 proofs can be used to determine when the Cryptol specification falls out of sync with the C implementation and needs
51 to be updated.
