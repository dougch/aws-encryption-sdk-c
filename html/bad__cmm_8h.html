◆ aws_cryptosdk_null_cmm()

TESTLIB_API struct aws_cryptosdk_cmm aws_cryptosdk_null_cmm ( )

Returns a zero size CMM. This is a CMM with virtual functions implemented but which has all NULL vtable pointers, to verify that the VF code does not call function pointers set to null.

Because the destroy function does not work, we return this by-value and rely on stack cleanup to free the memory associated with this CMM.

◆ aws_cryptosdk_zero_size_cmm()

TESTLIB_API struct aws_cryptosdk_cmm aws_cryptosdk_zero_size_cmm ( )

Returns a zero size CMM. This is a CMM with virtual functions implemented but which has its size set to zero, which should prevent the virtual functions from ever being called. Because the destroy function does not work, we return this by-value and rely on stack cleanup to free the memory associated with this CMM.
